# my-first-project

Репозиторий содержит два самостоятельных HTML-проекта: лендинг продукта **СамПродал** и отдельная личная визитка. Каждый — один файл без зависимостей и фреймворков, готов к публикации «как есть» на любом хостинге или GitHub Pages.

---

## Проекты

### 1. СамПродал — лендинг продукта (`landing.html`)

Telegram-бот, который превращает сырую домашнюю фотографию в готовое объявление: заменяет фон на стильный интерьер (не искажая саму вещь), пишет продающий SEO-текст и формирует полноценный лот за пару минут. Ключевое отличие от аналогов — бот честен с покупателями: реальное состояние товара сохраняется, поэтому продажи идут без возвратов и негатива.

#### Для кого
- Реселлеры и продавцы б/у вещей
- Мастера ручной работы, авторы декора и кастомного мерча
- Все, кто выкладывает много товаров и хочет тратить на это минуты, а не часы

#### Страницы / блоки, которые уже есть
| Файл | Содержимое |
|---|---|
| `landing.html` | Hero + 3 карточки преимуществ + блок «для кого» + CTA × 2 |

#### Страницы, которые планируются
| Страница | Описание |
|---|---|
| Примеры работ / портфолио | До/после — фото и текст, которые сделал бот |
| FAQ | Ответы на частые вопросы пользователей |
| Блог / статьи | Советы по продажам, про ИИ и упаковку товаров |

### 2. Личная визитка (`vizitka.html`)

Отдельный самостоятельный проект — визитная карточка Ксении Чудиновой как промпт-инженера и AI-разработчика. Одностраничный сайт с кратким описанием специализации, тегами навыков и кнопкой «Написать в Telegram».

---

## Дизайн-система

Оба проекта разработаны в едином визуальном стиле:

- **Фон:** тёмный (`#0d0d14` / `#0a0a0a`) с градиентными пятнами-блобами (blur) в углах
- **Акценты:** фиолетовый `#a259ff`, красный/розовый `#ff6b6b`, голубой `#00d4ff`, зелёный `#43e97b`
- **Текст:** градиентные заголовки через `background-clip: text`
- **Компоненты:** карточки с `border-radius: 20–28px`, тонкая граница `rgba(255,255,255,0.07)`, hover с подъёмом `translateY(-4px)`
- **Анимации:** CSS-only — пульс на бейдже, scroll-reveal через IntersectionObserver, плавающие частицы
- **Типографика:** системный стек `-apple-system, BlinkMacSystemFont, 'Segoe UI'`
- **Адаптив:** мобильный через `@media (max-width: 600px)`, fluid-размеры через `clamp()`
